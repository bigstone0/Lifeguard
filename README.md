![](../header.png)

## 실행 방법

웹
```sh
cd "프로젝트 위치/lifeguard/lifeguard_server/src/main"
npm start
```

앱
```sh
cd "프로젝트 위치/lifeguard/lifeguard_app"
react-native run-android
```
