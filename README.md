## 소개

항만 재난 대피소인 라이프가드웨 개폐 상태 및 내부 상태를 확인할 수 있고 개폐를 조작할 수 있습니다.


## 실행 방법

웹
```sh
cd "프로젝트 위치/lifeguard/lifeguard_server/src/main"
npm start
```

앱
```sh
cd "프로젝트 위치/lifeguard/lifeguard_app"
react-native run-android
```
